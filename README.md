# ⛽ India Petrol Price Analysis (2000–2026)

## Project Overview

This Data Analytics project explores the historical trend of petrol prices in India from 2000 to 2026 using Python.

The objective is to analyze long-term price movements, identify trends, and visualize petrol price changes over time using data analytics techniques.

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Dataset

The dataset contains yearly petrol prices in India from 2000 to 2026.

### Features

* Year
* Petrol Price (₹/Litre)

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insight Generation

## Visualization

The project includes a line chart showing the trend of petrol prices in India over the years.

## Key Insights

* Petrol prices increased from approximately ₹26/L in 2000 to over ₹100/L in 2026.
* Significant growth was observed during 2021–2022.
* Despite temporary declines in some years, the long-term trend remains upward.
